Interface Tool


Tool for build common.lvl with your custom tga/msh/odf/fx

How to use:

get original tga/msh/odf/fx in folder "Data\Common\", edit and save in "Custom\Common\"

don't copy all original files in "Custom\Common\" only ones wich you want to replace

launch common.bat for build common.lvl in folder "_LVL_PC\Common\ENG\"
